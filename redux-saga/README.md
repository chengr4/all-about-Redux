# Redux-Saga

## Sagas

- A Saga is a Collection of Sub-Transactions (T1, T2, T3, ..., Tn)
- Each Sub-Transaction has a Compensating Transaction 

> Compensating Transaction (Request) must be idempotent (一直 retry 直到成功為止 )
